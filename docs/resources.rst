Resources
=========
This section describes classes and functions private to <application-name>. It is intended to document how the application works

labs.lab-1
----------
Introduction lab covering basic repository setup and code examples. Includes a code golf solution for counting prime numbers.

labs.lab-2
----------
Covers Python package management with pip and virtual environment setup. Students learn to create isolated development environments and manage project dependencies.

labs.lab-3
----------
Introduces database connectivity using PostgreSQL, Flask, and SQLAlchemy. Students create database schemas, define models, and establish relationships between tables.

labs.lab-4
----------
Compares SQL queries with ORM operations using SQLAlchemy. Students implement database operations using both raw SQL and object-relational mapping techniques.

labs.lab-5
----------
Focuses on building an intentionally bad user interface with Flask. Students create multiple webpages with navigation, styling, and images to understand UI/UX principles.

labs.lab-6
----------
Implements user authentication with login functionality. Students create forms that validate user credentials against a database and route users based on authentication results.

labs.lab-8
----------
Covers unit testing with pytest and pytest-flask. Students write test cases to verify application functionality and learn test-driven development practices.

labs.lab-9
----------
Introduces documentation generation using Sphinx. Students set up documentation structure, create reStructuredText files, and deploy documentation to GitHub Pages.

labs.lab-10
----------
Covers containerization with Docker. Students deploy PostgreSQL and Flask containers using Docker Compose, creating isolated development and deployment environments.
